# 🪟 Windows Installation

✅ Recommended: download and run the installer directly.

## 1. Download the installer

- Mainland China:
  - https://www.qidong.ai
  - https://www.starter.top
  - https://www.aistarter.cc

- Global:
  - https://www.starter.one

Example filename: `AIStarter-Setup-5.2.0.exe`

## 2. Install steps

1. Download the `.exe`
2. Double‑click the installer
3. Follow the wizard to finish
4. On first launch, set the **project root** (stores AI projects)

> Directory naming rules: use CamelCase (e.g., `MyAIProjects`), consisting of letters, digits, and underscores only. Do not use Chinese, spaces, or special characters (`#`, `&`, `!`, `()`, spaces, etc.).
>
> Important: Do not set the program install directory to the same path as the project root. Installing, updating, or uninstalling could otherwise delete your downloaded projects, models, and plugins.
