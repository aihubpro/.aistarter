# 🪟 Windows 安装

✅ 推荐方式：直接下载安装包

## 1. 下载安装包

- 国内官网：
  - [https://www.qidong.ai](https://www.qidong.ai)
  - [https://www.starter.top](https://www.starter.top)
  - [https://www.aistarter.cc](https://www.aistarter.cc)

- 国际站：
  - [https://www.starter.one](https://www.starter.one)

示例文件名：`AIStarter-Setup-5.2.0.exe`

## 2. 安装步骤

1. 下载 `.exe` 文件；
2. 双击运行安装包；
3. 按照向导点击“下一步”，完成安装；
4. 首次启动时，需设置 **项目根目录**（用于存放 AI 项目文件）。

> ⚠️ 目录命名规范：项目根目录名称应采用驼峰命名（如 `MyAIProjects`），仅支持大小写英文字母、数字和下划线（_），禁止使用中文、空格及任何特殊字符（如 `#`、`&`、`!`、`()`、空格等）。
>
> ⚠️ 重要提醒：请勿将软件安装目录与项目根目录设置为同一路径！若两者位于同一文件夹，更新或卸载程序时可能导致已下载的 AI 项目、模型、插件等数据被一并删除。
